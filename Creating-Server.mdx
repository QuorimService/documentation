# Create a Server

To use Quorim, you must first create a server and connect it to your **Roblox PRC (Police Roleplay Community)** and **Discord server**.

Quorim relies on PRC’s official API to collect analytics.  
Before starting, make sure you have access to the **PRC API Pack**.

<Card
  title="PRC API Requirement"
  icon="key"
  iconType: "duotone"

  horizontal
>
  You must own the PRC API Pack for analytics to function.
</Card>

---

## Requirements

Before creating a server, ensure the following:

- You own or have access to a **PRC API Key**
- You are an **Owner**, **Administrator**, or have **Manage Server** permission in the Discord server
- The Quorim bot is invited to your Discord server

---

## How to Create a Server

<Steps>

### Log in to Quorim

Go to:
https://quorim.xyz/login

Log in using your Discord account.  
Once logged in, you will be redirected to your dashboard.

---

### Open the Dashboard

Visit:
```

/dashboard

```

On the dashboard, click the **Create Server** button.  
You will be redirected to the server creation flow at:
```

/create-server

```

---

### Enter Server Name

Enter a **unique server name** for your PRC server.

This name:
- Identifies your server inside Quorim
- Cannot be duplicated
- Can be changed later if needed

Once entered, click **Next**.

---

### Link Your Discord Server

Select the Discord server you want to link.

Requirements:
- You must be the **Owner**, **Administrator**, or have **Manage Server** permission
- The Quorim bot must be invited to the server

If you just invited the bot:
- Click the **Refresh** button to reload available servers

Once the Discord server is linked, click **Next**.

---

### Enter PRC API Key

Paste your **PRC API Key** into the input field.

- If the API key is **valid**, you can proceed to the next step
- If the API key is **invalid**, you will not be able to continue

Make sure the API Pack is active on your PRC account.

---

### Select Timezone

Your timezone is automatically selected based on your **IP address**.

Quorim uses your IP only for timezone detection.  
Learn more in our privacy policy:
https://quorim.xyz/privacy

You can manually change the timezone if needed.

Once selected, click **Next**.

---

### Confirm Server Details

You will see a confirmation summary containing:
- Server name
- Linked Discord server
- Selected timezone
- First 4 characters of your PRC API key

Review the details carefully.

To continue:
- Agree to the **Terms of Service**
- The **Continue** button will appear once accepted

Click **Continue** to finalize setup.

---

### Server Created

Your server will now be created and added to your dashboard.

You can manage analytics, players, and settings directly from the dashboard.

</Steps>

---

## After Creation

Once your server is created:
- Initial data processing may take time for large PRC servers
- Analytics populate gradually to avoid rate limits
- Data updates automatically as new activity is processed
